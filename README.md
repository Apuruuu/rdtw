# 日大体温填写辅助RDTW 1.0
自动填写日大的体温登录系统

需要的操作：
- 打开网页
- 登录你的账号
- 等待自动填写（根据网络状况，3~10秒不等）
- 关闭网页

安装方法：
- clone本程序
- 启动谷歌浏览器
- 进入扩展程序 chrome://extensions/
- 启用**开发者模式**
- 点击**加载已解压的扩展程序**
- 选择刚才clone的本程序根目录
- **启用**日大填体温 1.0

停用及卸载：
- 启动谷歌浏览器
- 进入扩展程序 chrome://extensions/
- 取消**启用**日大填体温 1.0
- 点击**移除**
- 删除本程序的目录
